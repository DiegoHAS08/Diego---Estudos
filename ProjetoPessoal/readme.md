Este é um simples gerenciador de tarefas que roda no terminal, desenvolvido em Python. Permite:

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

Como usar

```bash
python main.py add "Estudar Python"
python main.py list
python main.py done 1
python main.py remove 2
